## ⚡ 台電電費分析系統

### 功能說明
- 📊 **用電量圖表**：柱狀圖顯示各期用電
- 💰 **電費分析**：圖表顯示各期電費
- 🔍 **多維度篩選**：按電號、年度、期別篩選
- 📋 **詳細資料表**：顯示每期明細

### 操作說明
1. 開啟網頁，預設顯示全部電號的總計
2. 在第一個下拉選單選擇電號，圖表自動更新
3. 切換「依年度」/「依期別」查看不同維度
4. 指定期別可顯示該期各電號的用電比較表

### 資料格式
```json
{a: "電號", p: "期別代碼", y: 西元年, k: 用電度數, c: 電費金額}
```
